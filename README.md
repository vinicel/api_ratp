**What's RATP?**
RATP is a state-owned public transport operator and maintainer headquartered in Paris, France, which includes underground, tramway, bus and RER.
Only Underground and RER were used for this project.

## Requirements

- Php 7
- Composer

## Installation

Install project

```bash
composer install
```

Start api
```bash
./bin/console s:r
```
